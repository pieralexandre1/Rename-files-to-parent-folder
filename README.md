# Rename files to parent folder
Rename files to parent folder

<img width="534" height="253" alt="image_of_app" src="https://github.com/user-attachments/assets/2042ea37-7dea-485c-9660-e40f7780f155" />

Lets you rename all the files of a chosen folder to their parent folder

You got a few options like 
-Top folder: Only add the parent folder at the very top of the chosen directory or the chosen directory folder when the checkbox is turned on.
-Append folder: Add each folder name from the top until the current folder that the files are in.
-Direct parent folder: Add only the folder name that the files are currently in.

Undo button will only let you reverse the last operation; be careful not to activate consecutive rename if you are not satisfied with the result.

Work with long path.

Work on windows 10 on my end, haven't try other windows version or os.
